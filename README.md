## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/mehrdadmmz/address-to-map.git
   cd address-to-map
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the app:
   ```sh
   npm start 
   ```
4. Open index.html in your browser to use the app.


## Requirements

Node.js (v14 or higher)
